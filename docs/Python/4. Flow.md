Condition             `if` `elif` `else`                  Decision-making
Loop                     `for` `while`                        Repeat action
Loop control        `break` `continue`              Control inside loops
Function               `def` `pass` `return`            Structure & reuse
Exception             `try` `except` `finally`      Handle errors without crashing


### If - elif - else
```python
if num > 0:
	print("Positive")
elif num == 0:
	print("Zero")
else:
	print("Negative")
```

#### `and` - `or` - `not`
```python
if (age > 18 and is_student) or is_admin:
	print("Access granted")
elif not something:
	print("Access granted")
else:
	print("Access denied")
```


### For - while
```python
for i in range(1,6):
	if i == 3:
		continue          #skip 3
	if i == 5:                 
		break             #exit loop
	print(i)
```

#### Infinity loop
```python
count = 0
while True:
	count += 1
	if count >= 5:
		break            #exit loop
```

#### _ Underscore
```python
#Looping a fixed number of times
for _ in range(3):
	print("Hello")

#Ignore value
for _, value in person.items():
	print("Value", value)
```
