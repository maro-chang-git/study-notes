
# Type of Errors
1. Syntax Errors
2. Runtime Errors
3. Logical Errors

### Syntax Errors
- Misspelled keywords
- Incorrect use of operator
- Omitting parentheses
- Not wrapping strings in single quotes or double quotes


```bash
    4 +
       ^
SyntaxError: invalid syntax

print("Big Python)
                     ^
SyntaxError: EOL while scanning string literal
>>>
```

### Runtime Errors
Occurs while running the program

```bash
  File "<stdin>", line 1, in <module>
ZeroDivisionError: division by zero
```

### Logical Errors
NO notification -> return wrong result
