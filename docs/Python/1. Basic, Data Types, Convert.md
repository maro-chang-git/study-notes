### Global variables
The variable that is ***defined outside of any function*** OR with `global`, and is ***accessible from any where in the file***. 
```python
x = 10             #Global variable
def show()
	x = 4
	print(x)       # x = 4

print(x)           # x = 10
```

Use `global` inside function
```python
count = 0

def increment():
	global count
	count += 1

increment()          #count = 1
increment()          #count = 2
```

üö´ ERORR: is don't use `global`
```python
x = 5
def change():
    x += 1  # ‚ùå UnboundLocalError unless you declare global x

change()
```

## Int, Float, Complex - Numeric Type
`int`         -     `5`            -    Integer       - Unlimited range
`float`     -     `3.14`      -    Float           - -
`complex` -     `2 + 3j`  -    Mixed type - is mainly used in advanced math (not common in everyday use)

```python
#Integer (int)
a, b = 10, 3
#OR 
a = 10, b = 3

print("Integer division:", a // b)   # 3
print("Modulus:", a % b)             # 1
print("Power:", a ** b)              # 1000
print("Type of a:", type(a))         # <class 'int'>
x = float(a)                         # convert to float

# Float
x = 5.5
y = 2.0
print("\nFloat division:", x / y)    # 2.75
print("Rounded:", round(x / y, 1))   # 2.8
print("Type of x:", type(x))         # <class 'float'>
a = int(x)                           # convert to int

# Precision and limits
big_float = 1.8e308
too_big = 1.9e308
print("\nBig float:", big_float)     # ~1.8e+308
print("Too big float:", too_big)     # inf (overflow)

# Complex
z1 = 2 + 3j
z2 = 1 - 1j
z3 = z1 * z2
print("\nComplex number z1:", z1)
print("Real part:", z1.real)         # 2.0
print("Imaginary part:", z1.imag)    # 3.0
print("Multiplication z1 * z2:", z3) # (5+1j)
print("Type of z3:", type(z3))       # <class 'complex'>
```

### Range
What is range
- It's a built-in data type that returns an ***immutable sequence of integer***
- It does not store all numbers in memory -> more efficient. 

```python
range(start, stop, step)
range(0, 100, 5)
```
- `start`: from 0
- `stop`: until < 100 (exclusive)
- `step`: jump 5 steps - default is 1


Examples:
```python
print(list(range(5)))               #start = 5                    [0,1,2,3,4]
print(list(range(2, 6)))            #start = 2, stop = 6          [2,3,4,5] 
print(list(range(10, 0, -2)))       #start = 10, stop = 0, step = -2       [10,8,6,4,2]

#Use in loop
for i in range(3):
    print(i)

for i in range(len(nums))
	print(nums[i])
```

## Mapping Type / Hash / Dictionary

`dict`    - `{"name": "Alice", "age": 25`      - Key-value pairs

A dictionary is:
- Unordered (before python 3.7), now ***ordered***
- Mutable
- ***Indexed by unique key***

#### Create 
```python
person = {
	"name": "John",
	"age": 36,
	"is_student": False
}

# Add new key-value
person["city"] = Sydney

#Get value from key
print(person["name"])            #John

#Update value
person["age"] = 25

#Remove item
person.pop("city")

#Empty dictionary
person.clear()
```

üö´ **Safe Access with `.get()`**
```python
print(person.get("email"))        # None (no error if key doesn't exists)
```

#### Loop
```python
#Print key -> value
for key in person:
	print(key, "->", person[key])   #name -> John

#Print values only
for val in person.values():         #return values object
	print(value)                    #["John", 36, False]

#Print keys only
for key in person.keys():           #return keys object
	print(key)                      #["name", "age", "is_student"]

```

##### Using items() -> Best use
```python
my_dict = {"name": "Alice", "age": 30, "city": "New York"}  

#Option 1: key-value
for key, value in my_dict.items():  
	print(f"{key}: {value}")

#Option 2: item[0] is key. item[1] is value
for item in person.items():
¬† ¬† print(f"{item[0]}: {item[1]}")

#Option 3: _ underscore to ignore unnessary values  ---> Best use

	print("Value", value)
```

### Extract unique value in list with condition
Problem:
- Extract unique value from list A
- Condition x > 0 (positive number)

```python
A = [[1, -10, 3, 6, -5, 4, 1, 2]]

#Option 1: Faster O(1)
positives = set(x for x in A if x > 0)

#Option 2: Slower O(n)
positives = set()
for num in A:
	if num > 0:
		positives.add(num)
```
