## Intermediate Level: Real Brain Yoga Starts

Feels like twisting letters into origami cranes.

6. **Group all anagrams in a list**  
    Classic LeetCode 49
    
7. **Find all anagram pairs**  
    For a list like `["ab", "ba", "cat", "act"]`
    
8. **Find all starting indices of substrings that are anagrams**  
    Sliding window time
    
9. **Detect if a string contains any anagram of a target**
    
10. **Count how many anagram groups exist**
    
11. **Return the largest anagram group**
    
12. **Return the smallest anagram group (size > 1)**  
    The lonely sneaky ones
    
13. **Check if phrase A can be rearranged to form phrase B**  
    (Multi-word anagram puzzle style)

## Advanced Level: Letter-gymnastics, now with sweat

Now letters do parkour across your brain.

14. **Find all unique anagram substrings**
    
15. **Count total unique anagram substrings**
    
16. **Longest substring that is an anagram of another substring**
    
17. **Find palindrome anagrams**  
    (Like `"carrace"` -> `"racecar"`)
    
18. **Check if a string can form _any_ palindrome**
    
19. **Anagram with wildcard characters**  
    `"a?b"` vs `"bca"`
    
20. **Anagram mutation cost**  
    Minimum edits to convert string A into an anagram of B
    
21. **Frequency signature hashing**  
    Build a hash function for anagram grouping
    
22. **Real-time streaming anagram detector**  
    (As letters drip in like a leaky faucet)
    
23. **Anagram ranking**  
    Given letters, find position of word in sorted anagram list
