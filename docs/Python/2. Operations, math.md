## Operations
### Arithmetic Operator 
Used for math on numbers
`+`        -          Addition                         -         `3 + 2`              - `= 5`
`-`        -          Subtraction                    -         `3 - 2`              - `= 1`
`x`        -          Multiplication                 -         `4 * 2`              - `= 8`
`/`        -          Division                          -         `3 / 2`              - `= 1.5`
`//`      -          Floor division                 -         `7 // 2`            - `= 3`
`%`        -          Modulus (Remainder)   -         `3 % 2`              - `= 1`
`**`      -          pow                                -         `2**3`                - `= 8`

### Assignment Operator
Used to assign and update values
`=`        -      Assign                           -        `x = 10`
`+=`      -      Add and Assign            -        `x += 10`
`-=`      -      Subtract and Assign    -        `x -= 10`
`*=`      -      Multiply and Assign     -        `x *= 10`
`/=`      -      Divide and Assign        -        `x /= 10`

### Identity
Used to compare object identity (not just value)
`is`                -         `x is y`                 -     True if x and y refer to same object
`is not`        -         `x is not y`         -     True if x and y are different object

## Math 
Python comes with a built-in `math` module. 
```python
import math
```

### Rounding & Absolute
`round(x)`                     - Round to nearest integer                - `round(3.6)` -> `4`
`abd(x)`                         - Absolute                                           - `abs(-2)` -> `2`
`math.floor(x)`            - Round down to nearest int             - `math.floor(3.9)` -> `3`
`math.ceil(x)`              - Round up to nearest int                  - `math.ceil(3.1)` -> `4`
`math.trunc(x)`            - Remove decimal part (truncate)     - `math.trunc(3.6)` -> `3`

### Power & Roots
`math.pow(x, y)`   -    `x ** y`   - x^y
`math.sqrt(x)`       - square root
`math.exp(x)`         - e raised to the x
